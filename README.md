# Apostrophe Backend Project

Backend server built with Express.js for user management system.

## Getting Started

### Installation

```bash
npm install
```

### Running the Server

```bash
npm start
```

The server will start on `http://localhost:3000`

### Testing

Visit `http://localhost:3000` in your browser or use Postman to test the endpoint.

## API Endpoints

### GET /
Returns a Hello World message.

**Response:**
```json
{
  "message": "Hello World",
  "status": "success"
}
```
